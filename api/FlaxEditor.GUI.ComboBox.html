    Remarks

    Difference between ComboBox and Dropdown is that ComboBox uses native window to show the items list while Dropdown uses a custom panel added to parent window. This means that Dropdown will work on all platforms that don't support multiple native windows (eg. Android, PS4, Xbox One).
